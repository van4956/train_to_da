# Train to DA

Статический веб-тренажёр для подготовки к собеседованиям на позицию DA / DS / ML.

Проект построен на основе Markdown-карточек из моего Obsidian, не требует сервера или базы данных.
Основная цель — системное повторение теории, активное вспоминание и симуляция интервью.


## Возможности

- Более 400+ вопросов по SQL, Python, ML, Алгоритмам, Статистике, A/B тестам и т.д.
- Фильтрация по темам и уровню сложности
- 4 режима повторения:
  - Каталог
  - Карточки
  - Блиц-проверка
  - Интервью (с LLM)
- Статический хостинг через GitHub Pages
- Генерация данных одной командой

## Сборка данных

Подразумевается что общая База Знаний с карточками уже собрана, и перенесена в папку database/.

Каждая карточка это отдельный .md файл формата:
```
Вопрос
---
Ответ
---
#lvl_1
```

Для генерации `data.json`:

```bash
python build.py
```

- читает все `.md`
- парсит вопросы, ответы, теги
- формирует структуру данных для фронта
