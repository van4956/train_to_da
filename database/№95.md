
Какой результат вернёт данный запрос?

```sql
WITH RECURSIVE tree AS (
	SELECT 
		id, 
		parent_id, 
		name, 
		1 AS depth, 
		name AS path
	FROM categories
	WHERE parent_id IS NULL
	UNION ALL
	SELECT 
		c.id,
		c.parent_id,
		c.name,
		t.depth + 1 AS depth,
		CONCAT(t.path, ' > ', c.name) AS path
	FROM categories c
	JOIN tree t ON c.parent_id = t.id
	)
SELECT * 
FROM tree 
ORDER BY path;
```

---

Это рекурсивный запрос (RECURSIVE CTE), вернёт полную иерархию категорий с глубиной и текстовым путём вида «Root > Subcategory».
