

Объясните разницу между Precision и Recall.

---

Precision (точность) - показывает количество правильно предсказанных единичек(`TP`) среди всех ПРЕДСКАЗАННЫХ единичек `(TP + FP)`. То есть `Precision = TP / (TP + FP)`

Recall (полнота) - показывает количество правильно предсказанных единичек(`TP`) среди всех ИСТИННЫХ единичек (`TP + FN`). То есть  `Recall = TP / (TP + FN)`

|            | true 1   | true 0 |             |
| ---------- | -------- | ------ | ----------- |
| **pred 1** | TP       | FP     | Precision   |
| **pred 0** | FN       | TN     |             |
|            | Recall   |        |             |
