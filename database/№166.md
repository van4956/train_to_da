

Как безопасно открывать и закрывать файлы в Python?

---

Для безопасной работы с файлами используют контекстный менеджер `with`. Он гарантирует, что файл будет автоматически закрыт, даже если во время работы возникнет ошибка.

```python
with open('data.txt', 'r') as f:
    data = f.read()
```
