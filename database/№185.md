

Как быстро оценить MDE в Python?

---

MDE (Minimum Detectable Effect) — минимальный эффект, который эксперимент может обнаружить при заданном уровне значимости и размере выборки.

При нормальном приближении MDE оценивают по формуле: $MDE = z_{α/2} · σ · √(2 / n)$


```python
from scipy import stats 
import math  

def mde(sigma, n, alpha=0.05):     
	return stats.norm.ppf(1 - alpha / 2) * sigma * math.sqrt(2 / n)
```

MDE используют для проверки, имеет ли эксперимент смысл при данном размере данных.
