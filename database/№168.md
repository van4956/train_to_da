

Как работает `zip_longest` из `itertools`?

---

`zip_longest` объединяет элементы нескольких итерируемых объектов по индексам и продолжается до самого длинного из них. Если элементы в одном из итерируемых закончились, вместо них подставляется значение `fillvalue`.

```python
zip_longest([1, 2], ['a'], fillvalue='-')  # → (1, 'a'), (2, '-')
```
