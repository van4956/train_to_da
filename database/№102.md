
Когда полезно использовать `USING` вместо `ON` в `JOIN`?

---

`USING` применяют, когда таблицы соединяются по колонке с одинаковым именем.  
Он упрощает синтаксис `JOIN`, автоматически убирает дублирующийся столбец из результата и снижает риск опечаток по сравнению с `ON`.

```sql
select *
from users
join orders using (user_id);
```
