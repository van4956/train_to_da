

Когда стоит предпочесть векторизованный Pandas API вместо `apply`?

---

Векторизованные операции в pandas выполняются на уровне C/NumPy и обрабатывают массивы целиком, поэтому они значительно быстрее, чем `apply` с Python-функцией по строкам. 

`apply` стоит использовать только тогда, когда логику невозможно выразить векторизованно, требуется сложная Python-логика для каждой строки
    
Во всех остальных случаях предпочтительны векторизованные операции.
