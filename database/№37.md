
Какие есть методы по работе с датами/временем?

---

`DATE()` — обрежет время до даты (самый простой способ, но только в PostgreSQL)

```sql
date(sent_at)  --возвращает date
```

`DATE_TRUNC()` — округление к началу произвольного периода (обычно используется вместе c оконными функциями)

```sql
date_trunc('month', sent_at)  --возвращает timestamp
```

`EXTRACT()` — вытащить часть даты (удобен для фильтрации и сравнений, но не для группировки по полной дате)

```sql
extract(year from sent_at)  --возвращает int
...
where extract(year from sent_at) = 2024
```

`TO_CHAR()` — форматирование даты (для вывода и отчётов, не рекомендуется для логики и GROUP BY)

```sql
to_char(sent_at, 'YYYY-MM-DD')  --возвращает str
```
