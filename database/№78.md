
Что такое window frame и чем отличается `ROWS BETWEEN` от `RANGE BETWEEN`?

---

Window frame задаёт набор строк, по которым считается оконная функция для текущей строки.  `ROWS BETWEEN` — считает окно по физическому количеству строк (строка выше / ниже). `RANGE BETWEEN` — считает окно по значению в `ORDER BY` (например, диапазон дат или чисел), может включать несколько строк с одинаковым значением.
