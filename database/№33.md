
Чем отличается LIKE от ILIKE и когда использовать каждый?

---

`LIKE / ILIKE` используется для поиска подстроки.  
`LIKE` — чувствителен к регистру  
`ILIKE` — нечувствителен к регистру

```sql
select item_name                   -- item_name = 'Red Sweater'
from table
where item_name like '%sweater%'   -- ❌ не найдёт
where item_name ilike '%sweater%'  -- ✅ найдёт
```
