## ТЗ: Glassmorphism для карточек каталога

### Цель

Сделать карточки полупрозрачными в стиле **glassmorphism (iOS / VisionOS)**:

* фон просвечивает
* контент остаётся читабельным
* дизайн минималистичный, без анимаций

---

### Область применения

* Все карточки в **режиме «Каталог»**
* (опционально) использовать тот же стиль в режимах:

  * «Карточки»
  * «Блиц»
  * «Интервью»

---

### Требования к визуалу

* Полупрозрачный фон карточки
* Размытие фона **за карточкой**
* Лёгкая цветовая тонировка (тёмно-синяя / графитовая)
* Скруглённые углы
* Тонкая светлая обводка
* Текст внутри **НЕ должен быть размытым**

---

### Технические требования

#### CSS (обязательно)

Использовать `backdrop-filter`:

```css
.card {
  background: rgba(20, 30, 50, 0.55);
  backdrop-filter: blur(12px) saturate(140%);
  -webkit-backdrop-filter: blur(12px) saturate(140%);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 14px;
}
```

Допустимые диапазоны (можно подбирать визуально):

* `blur`: **8–16px**
* `alpha`: **0.45–0.65**
* `saturate`: **120–160%**

---

### Ограничения

* ❌ Не использовать JS
* ❌ Не добавлять анимации
* ❌ Не менять структуру HTML
* ❌ Не влиять на Markdown / KaTeX / подсветку кода

---

### Совместимость

* Chrome / Edge / Safari — обязательно
* Firefox — допустима деградация (без blur, только полупрозрачный фон)

---

### Критерии приёмки

* Карточка визуально «стеклянная»
* Фон читается сквозь карточку
* Текст внутри легко читается
* На Firefox интерфейс не ломается
