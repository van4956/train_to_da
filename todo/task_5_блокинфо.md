## ТЗ: Блок счётчика карточек (X из Y) с детализацией по уровням

### Цель

Сделать единый информационный блок, отображающий:

* количество карточек по текущим фильтрам
* соотношение **X из Y**
* распределение по уровням сложности
  в визуальном стиле карточек (прозрачность, центрирование).

---

### Текущее состояние

* Отображается только:

  * количество карточек после фильтрации
  * счётчики Easy / Medium / Hard
* Информация визуально «висит» в воздухе
* Нет связи с общим объёмом базы

---

### Требуемое поведение

#### Логика подсчёта

* **Y** — общее количество карточек в базе
* **X** — количество карточек после применения фильтров:

  * поиск
  * тема
  * уровень
* Отдельно отображать:

  * количество Easy
  * количество Medium
  * количество Hard
    (только среди X)

---

### Формат отображения (пример, не текст-обязательство)

```
2 из 39 карточек       0 Easy · 2 Medium · 0 Hard
```

---

### UI / Layout требования

#### Контейнер

* Отдельный блок под фильтрами
* Ширина:

  * строго как у карточек
  * центрирован по горизонтали
* Отступы:

  * визуально отделён от фильтров и карточек

#### Стиль

* Прозрачность:

  * аналогична карточкам
* Фон:

  * полупрозрачный
  * без жёстких границ
* Скругления:

  * такие же, как у карточек

---

### Типографика

* Текст компактный
* Без жирных акцентов
* Не перетягивает внимание с карточек
* Хорошая читаемость на фоне

---

### Поведение

* Блок обновляется **динамически** при:

  * изменении фильтров
  * изменении поискового запроса
* Если X = 0:

  * блок остаётся видимым
  * отображает `0 из Y карточек`

---

### Ограничения

* ❌ Не добавлять новые фильтры
* ❌ Не менять существующую логику фильтрации
* ❌ Не использовать отдельные попапы / тултипы
* ❌ Не дублировать информацию в других местах

---

### Критерии приёмки

* Видно соотношение X из Y
* Счётчики уровней соответствуют фильтрам
* Блок центрирован и визуально совпадает с карточками
* Прозрачность и стиль единообразны
